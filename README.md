# Intro-to-ML Coursework

This repo contains an R workflow for a lagged linear regression project on India's railways sector. The target is `real_gdp_growth_of_railways`, and the analysis uses lagged railway demand and investment variables to keep the model aligned with a time-series prediction task while staying inside a linear regression framework.

The main deliverable is the R script. A small Python fallback script is also included because the local machine used for this repo could not complete an R installation during setup, so the reference outputs in `outputs/` and `docs/report.md` were generated with the same lagged OLS logic through the fallback route.

## Structure

- `data/processed/railways_master.csv`: cleaned annual dataset extracted from the provided Excel workbook
- `scripts/railways_lag_linear_regression.R`: EDA, feature engineering, model comparison, and forecast script
- `scripts/generate_reference_outputs.py`: fallback output generator that mirrors the same specification when R is unavailable
- `outputs/`: generated summaries, plots, predictions, and forecast tables
- `docs/report.md`: short report generated by the R script

## How to run

From the repo root:

```bash
Rscript scripts/railways_lag_linear_regression.R .
```

Fallback output regeneration:

```bash
python3 scripts/generate_reference_outputs.py
```

## What the script does

1. Loads the cleaned railways dataset
2. Builds growth-rate predictors from key railway operating and investment series
3. Creates one-period lag variables
4. Runs exploratory data analysis and saves charts/tables
5. Compares a few parsimonious lagged linear regression specifications
6. Fits the best-performing model and produces a 2024-25 forecast if the required lagged predictors are available
